## 📖 Deskripsi
Project ini merupakan aplikasi berbasis C# yang mengimplementasikan koneksi ke database menggunakan MySQL.  
Aplikasi ini dibuat untuk memenuhi tugas mata kuliah yang berkaitan dengan pemrograman dan database.

## 🛠️ Tools & Teknologi
- C#
- .NET Framework
- MySQL
- Visual Studio

## ▶️ Cara Menjalankan Program
1. Buka project di Visual Studio.
2. Pastikan MySQL sudah terinstall.
3. Sesuaikan konfigurasi koneksi database.
4. Jalankan program dengan tombol Start.
